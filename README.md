# Telegram Profanity Analyzer

Мощный инструмент на Python для анализа экспортов Telegram-чатов и поиска нецензурной лексики, оскорблений и различных вариантов мата.

Скрипт обрабатывает JSON-экспорт Telegram, анализирует сообщения выбранных пользователей и формирует подробную статистику по использованию ненормативной лексики.

## Возможности

* Поиск русскоязычного мата и оскорблений
* Поддержка транслита
* Распознавание замен символов:

  * `а → @, 4`
  * `е → 3`
  * `о → 0`
  * и другие варианты
* Обнаружение растянутых слов:

  * `бляяяяять`
  * `суууука`
  * `хуууй`
* Поддержка смешанных раскладок клавиатуры
* Выбор одного или нескольких пользователей для анализа
* Подробная статистика по каждому участнику
* Красивый цветной вывод в консоль
* Экспорт отчётов в TXT
* Работа с большими Telegram-чатами

## Как работает

1. Экспортируйте Telegram-чат в формате JSON.
2. Поместите файл рядом со скриптом или укажите путь вручную.
3. Выберите пользователей для анализа.
4. Скрипт просканирует все сообщения.
5. Мат определяется с помощью набора сложных Regex-паттернов.
6. Формируется статистика и отчёт.

## Результаты анализа

Инструмент показывает:

* Количество матов у каждого пользователя
* Общую статистику по чату
* Даты и время обнаруженных сообщений
* Подробные TXT-отчёты

В основе проекта лежит большая коллекция регулярных выражений, которая умеет находить:
